# Student Management System

A simple Python command-line project to manage student records.

## Description

This project allows users to add, view, search, update, and delete student details using a menu-based system. Student data is stored temporarily in a Python list while the program is running.

## Features

- Add student details
- View all students
- Search student by ID
- Update student details
- Delete student record
- Exit the program

## Student Details Stored

Each student record contains:

- Student ID
- Name
- Age
- Mark
